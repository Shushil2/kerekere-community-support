### Requirements
- Build a Kerekere platform for Fijians to ask for and offer help
- Enable users to post requests for assistance (kerekere)
- Allow community members to respond and offer help
- Categories: financial, material goods, services, emotional support
- User authentication and profiles
- Request tracking and status updates
- Community-focused, culturally appropriate design

### Designs
- Warm, welcoming color scheme reflecting Fijian culture
- Clean, accessible interface for all age groups
- Mobile-first responsive design
- Clear request cards with category badges
- Easy-to-use request creation form
- Community dashboard showing active requests
